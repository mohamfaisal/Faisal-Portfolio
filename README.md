## 🚀 About The Project

This repository contains the source code for my professional portfolio. It is designed to be more than just a resume; it is a live demonstration of my ability to build immersive, high-converting digital experiences. 

The architecture focuses on clean code, seamless animations, and a dark-themed, cinematic aesthetic tailored for modern web standards.

### ✨ Premium Features Integrated:
* **Interactive UI/UX:** Features advanced glassmorphism, 3D hover physics (`vanilla-tilt.js`), and liquid-fill micro-interactions.
* **Custom Animation Sequencing:** Includes a bespoke staggered letter-reveal loader and glowing vertical experience timelines.
* **Performance Optimized:** Engineered for fast load times with optimized asset compression and responsive masonry grids.
* **Dynamic Media Galleries:** Custom-built smart lightboxes for high-resolution AI photography and cinematography displays.

* **Frontend:** HTML5, CSS3 (Custom Variables & Animations), Vanilla JavaScript, jQuery.
